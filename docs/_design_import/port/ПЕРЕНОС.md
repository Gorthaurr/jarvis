# Перенос дизайна «Премиум-минимал» в приложение

Чистый рестайл — структуру `index.html` и логику `renderer.ts` менять не нужно,
все имена классов совпадают 1:1.

## 1. CSS (обязательно)
Заменить файл:
```
apps/client/renderer/styles.css   ←   port/styles.css
```

## 2. Шрифт Manrope (рекомендуется)
Дизайн построен на **Manrope**. Положите вариативный woff2 рядом со шрифтами:
```
apps/client/renderer/fonts/Manrope.woff2
```
Скачать: https://fonts.google.com/specimen/Manrope → «Get font» → экспорт woff2
(вариативный, веса 200–800).
> Если файл не добавить — интерфейс гладко откатится на встроенный Inter,
> всё останется рабочим, просто без Manrope. Гугл-шрифты по ссылке не подойдут:
> CSP приложения (`default-src 'self'`) разрешает только локальные ресурсы.

## 3. Цвет живой волны (1 строка)
Волна на канвасе рисуется в коде. Замените золото/фиолетовый на циан:
```
apps/client/renderer/renderer.ts : ~407
- cctx.strokeStyle = currentState === "speaking" ? "#a78bfa" : "#f4ac5e";
+ cctx.strokeStyle = "#5ed6ff";
```
И заголовок окна (фон) под палитру, если хотите идеального совпадения:
```
apps/client/main/index.ts : backgroundColor: "#0b0d12"  →  "#0a0b0e"
```

## Что НЕ входит в этот рестайл
- **Вкладки в настройках** (Общее / Навыки / Ключи / Оплата) — это из макета;
  чтобы добавить их в реальное приложение, нужна небольшая правка `renderer.ts`
  (состояние активной вкладки + показ/скрытие групп) и `index.html`
  (таб-бар + контейнеры). Скажите — пришлю готовый патч.

Палитра, типографика, кнопки, поля, модалки, статус-бар, индикатор состояний —
всё уже в CSS и подхватится сразу после замены файла.
